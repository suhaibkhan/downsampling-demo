Timeseries Data Downsampling Algorithms Comparison

Raw (unsampled) data.
In random downsampling method, data is divided in to specified number of buckets and a random item is picked from each bucket to form the sampled data.
In average downsampling method, data is divided in to specified number of buckets and average of all items in each bucket is combined to form the sampled data.
In sum downsampling method, data is divided in to specified number of buckets and sum of all items in each bucket is combined to form the sampled data. This downsampling method changes the y axis.
In combined minimum-maximum downsampling method, data is divided in to specified number of buckets and from each bucket minimum and maximum data points (two values) is picked to form the sampled data.
Largest-Triangle-Three-Buckets algorithm proposed by Sveinn Steinarsson. For more details visit https://github.com/sveinn-steinarsson/flot-downsample
